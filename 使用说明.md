## 零、准备工作（只需做一次）

### 你需要有的东西：

| 条件 | 说明 |
|------|------|
| Windows 10/11 | 不支持 Mac/Linux |
| Python 3.10+ | [python.org](https://www.python.org/downloads/) 下载，安装时勾选 "Add Python to PATH" |
| NVIDIA 显卡 | 2GB+ 显存，需安装 [CUDA 驱动](https://www.nvidia.com/download/index.aspx) |
| 管理员权限 | 电脑管理员账户 |

### 0.1 安装 Interception 驱动 ⚠️ 必装

这是控制键鼠的内核驱动，**不装就跑不了**：

1. 下载 [Interception.zip](https://github.com/oblitum/Interception/releases/latest)
2. 解压到任意文件夹
3. **以管理员身份打开 cmd**，进入解压目录
4. 执行：
   ```
   install-interception.exe /install
   ```
5. **重启电脑**

> 验证：cmd 输入 `sc query interception`，显示 RUNNING 即成功

## 一、快速开始（3 步）

### 第一步：克隆或下载项目

```bash
git clone https://github.com/SCH-CMYK/game-automation.git
cd game-automation
```

不会用 git？直接下载 ZIP：[点此下载](https://github.com/SCH-CMYK/game-automation/archive/refs/heads/main.zip)，解压到任意文件夹。

### 第二步：一键安装

**右键 `setup.bat` → 以管理员身份运行**

自动完成：
- 检查 Python 版本
- 创建虚拟环境
- 安装所有依赖（使用清华镜像，国内下载飞快）
- 自动下载模型和地图文件
- 检查 Interception 驱动状态

### 第三步：启动

双击 `run.bat`（自动请求管理员权限）

## 二、如果安装失败

### 问题 1：提示 "Python not found"

→ 没装 Python。去 [python.org](https://www.python.org/downloads/) 下载安装，**一定要勾选 "Add Python to PATH"**

### 问题 2：pip 安装报错

→ 重试。如果反复失败，手动执行：
```bash
.venv\Scripts\activate
pip install -r requirements.txt
```

### 问题 3：模型下载失败

→ 网络问题。手动执行：
```bash
python download_models.py
```
或者从 [Releases 页面](https://github.com/SCH-CMYK/game-automation/releases/tag/v1.0) 手动下载 3 个文件放入对应目录。

### 问题 4：提示 Interception 驱动未安装

→ 回到「第零步」安装驱动，记得**重启电脑**。

### 问题 5：启动后提示 CUDA 不可用

→ 没装 NVIDIA 显卡驱动或 CUDA。安装 [CUDA Toolkit](https://developer.nvidia.com/cuda-downloads)，或者改用 CPU 版 PyTorch（速度慢很多）。

## 三、界面说明

启动后看到三个标签页：

### 侧边栏（固定）
- **模型加载区**：选择模型 → 点击加载
- **采矿控制区**：开始/暂停/停止采矿
- **快捷键提示区**：显示可用热键

### 标签页 1：检测预览
- 实时显示游戏画面，FPS 显示在右上角

### 标签页 2：模型训练
- 数据集管理、YOLO 模型训练、数据标注工具

### 标签页 3：路线规划（核心功能）
- 大地图自动显示、途经点编辑、导航控制

## 四、自动导航使用教程

### 4.1 第一次使用

1. **打开游戏**，进入游戏画面
2. **双击 `run.bat`** 启动程序
3. 侧边栏加载 YOLO 模型（选择 `best_20260601.pt` → 加载模型）
4. 切换到「路线规划」标签页
5. 大地图自动加载显示

### 4.2 设置小地图

- **自动检测**：程序用霍夫圆自动检测右上角圆形小地图
- **手动框选**：检测不准时，点击「框选小地图」手动选择

### 4.3 编辑路线

| 操作 | 方式 |
|------|------|
| 添加途经点 | 左键点击地图 |
| 删除途经点 | 右键点击 |
| 缩放地图 | 鼠标滚轮 |
| 移动地图 | 拖拽 |
| 保存路线 | 点击「保存路线」 |

> 途经点每隔 80-120 像素一个即可，不需要很密集

### 4.4 开始导航

1. 确保已加载路线（地图上有途经点和连线）
2. 点击「开始导航」

程序自动：定位 → 走路 → 采矿 → 战斗逃跑 → 下一途经点

**紧急停止**：`Ctrl + Shift + Q`

## 六、常见问题

**Q: 启动后不走路？**
→ 检查是否加载了 YOLO 模型、是否加载了路线、小地图区域是否正确

**Q: 采矿不准？**
→ 调整 `src/utils/config.py` 中的 `mine_hold_duration` 和 `mine_cooldown`

**Q: 被封号？**
→ 本工具只做键鼠模拟，不修改游戏内存。但仍存在风险，请自行评估

**Q: 想训练自己的模型？**
→ `Ctrl+T` 收集数据 → 模型训练标签页 → 标注工具 → 开始训练

## 七、项目结构

```
game-automation/
├── setup.bat              ← 一键安装（右键管理员运行）
├── run.bat                ← 一键启动
├── download_models.py     ← 模型下载工具
├── main.py                ← GUI 主程序
├── src/
│   ├── engine/            ← 底层引擎（采集/视觉/输入/定位）
│   ├── automation/        ← 自动化逻辑（采矿/导航/传送）
│   ├── training/          ← 模型训练
│   └── utils/             ← 工具集
├── models/                ← 模型文件（运行 download_models.py 下载）
├── maps/                  ← 大地图
├── routes/                ← 路线文件
├── tests/                 ← 测试（48 项）
└── logs/                  ← 日志
```
