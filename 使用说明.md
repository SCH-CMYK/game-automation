## 一、懒人包使用（一键配置 + 一键启动）

### 第一步：安装环境

右键 `setup.bat` → **以管理员身份运行**

这会自动完成：
- 检查 Python 环境
- 升级 pip
- 安装所有依赖包

### 第二步：启动程序

双击 `run.bat` 即可启动（自动请求管理员权限）

> **首次使用需要：**
> 1. 确保 `models\` 目录下有模型文件
> 2. 确保 `maps\big_map.png` 大地图文件存在

## 二、界面说明

启动后看到三个标签页：

### 侧边栏（固定）
- **模型加载区**：选择模型 → 点击加载
- **采矿控制区**：开始/暂停/停止采矿
- **快捷键提示区**：显示可用热键

### 标签页 1：检测预览
- 实时显示游戏画面
- FPS 显示在右上角

### 标签页 2：模型训练
- 数据集管理
- YOLO 模型训练
- 数据标注工具

### 标签页 3：路线规划（核心功能）
- 大地图自动显示
- 途经点编辑
- 导航控制

## 三、自动导航使用教程

### 3.1 第一次使用

1. **打开游戏**，进入游戏画面
2. **双击 `run.bat`** 启动程序
3. 在侧边栏加载 YOLO 模型（选择 `best_20260601.pt` → 点击加载模型）
4. 切换到「路线规划」标签页
5. 大地图会自动加载显示

### 3.2 设置小地图

程序需要知道小地图在屏幕上的位置：

- **自动检测**：程序会自动用霍夫圆检测小地图位置（右上角圆形区域）
- **手动框选**：如果自动检测不准，点击"框选小地图"按钮，在弹窗中拖动选择

小地图位置会保存在 `minimap_region.json` 中，下次启动自动加载。

### 3.3 编辑路线

在路线规划标签页的大地图上：

| 操作 | 方式 |
|------|------|
| 添加途经点 | 左键点击地图 |
| 删除途经点 | 右键点击 |
| 缩放地图 | 鼠标滚轮 |
| 移动地图 | 拖拽 |
| 保存路线 | 点击「保存路线」 |
| 加载路线 | 点击「加载路线」 |

**路线提示**：
- 途经点不需要非常密集，每隔 80-120 像素一个即可
- 途经点只是用于标记行走路线，不代表该位置有矿石
- 矿石会在路途中自动检测和采集

### 3.4 开始导航

1. 确保已加载路线（地图上能看到途经点和连线）
2. 点击「开始导航」

程序会自动：
- LoFTR 实时定位你在大地图上的位置
- 控制角色走向下一个途经点
- 途中检测到矿石自动采集
- 检测到战斗自动退出
- 到达最後一个途经点自动停止

**紧急停止**：按 `Ctrl + Shift + Q`

## 四、自动采矿说明

### 4.1 采矿时机

- **路途中**：每 0.4 秒检测一次画面，发现矿石自动采集
- **每个途经点**：到达后检测一次，有矿就采

### 4.2 采矿流程

1. 检测到矿石 → 暂停导航
2. 自动瞄准矿石位置
3. 点击鼠标左键丢出精灵
4. 等待 5 秒采集动画
5. 检测是否还有矿 → 有就继续采
6. 屏幕没矿了 → 恢复导航继续走

### 4.3 采矿参数调整

编辑 `config.py` 中的 `MiningConfig`：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `mine_hold_duration` | 2.0 | 鼠标按住时长（秒） |
| `mine_cooldown` | 6.0 | 两次采矿间隔（秒） |

## 五、战斗逃跑说明

### 5.1 检测方式

每 0.2 秒用霍夫圆检测小地图是否消失。

- 小地图消失 → 判定为进入战斗
- 等待 2 秒（战斗加载动画）
- 自动按 `ESC` → 点击「是」退出战斗
- 退出后跳到下一个途经点继续导航

### 5.2 退出战斗按键

游戏中退出战斗的确认界面：
- 按 `ESC` 弹出退出确认框
- 「是」按钮位置已校准为 (1114, 883)
- 如果按钮位置不准，运行 `check_mouse.py` 重新校准

## 七、常见问题

### Q1: 启动后不走路？

检查：
1. 是否加载了 YOLO 模型
2. 是否加载了路线
3. 小地图区域是否正确

### Q2: 采矿不准确？

调整 `config.py` 中的参数：
- 增大 `mine_hold_duration`（采更久）
- 调整检测置信度

### Q3: 自动换角色被检测到？

- 本工具使用 Interception 驱动（内核级键鼠模拟），不修改游戏内存
- 但仍存在被封号风险，请自行评估

### Q4: 缺少模型文件？

确保以下文件存在：
- `models/best_20260601.pt` — YOLO 检测模型
- `models/loftr_model.onnx` — LoFTR 定位模型
- `maps/big_map.png` — 大地图

### Q5: 想训练自己的模型？

1. 用 `Ctrl+T` 计时截图收集数据
2. 切换到「模型训练」标签页
3. 点击「启动标注工具」标注数据
4. 设置参数 → 点击「开始训练」
